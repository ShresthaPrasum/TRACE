# TRACE

A 2D physics puzzle game where the player leaves it's trace by dashing in speed of light. The trace copies the player movement for 2-3 seconds.

## Game Mechanics
- Player dashes using Q Key and leaves a trace of it.
- Traces can't collide with the actual player but can collide with other traces and objects. Trace body feels like quick sand to the player.
- Traces can trigger colliders.
- Traces get's freeze after 2 seconds of activation
- If the traces touch any killer object the level restarts
- 10 playable levels with menu pages.

## Goal
- Make the player enter the portal

## Control
- W for jump
- A and D for horizontal movement
- Q for Dash
